## Summary

- **KNOWLEDGE & AI admin section** added to the admin sidebar below PEOPLE & PERFORMANCE, with four subsections:
  - **AI Hub:** Dashboard, AI Agents, Agent Analytics, Agent Categories, Prompt Templates, Email Drafting, Deal Coaching

  - **User Memory:** Memory Dashboard, User Memory Stats, Search Analytics, Team Learning Patterns
  - **Knowledge Base:** Common Knowledge, Processing Queue, Sources, Categories, Batch Upload, Files, Sync Status, Gemini RAG

- **AI Features dashboard** at `/admin/ai` with:

  - Quick stats: Active AI Agents, AI Executions, Summaries Generated, AI Tokens Used (from `useAIDashboardStats`)
  - “About AI Features” info card

- **AI Agents inside admin:** “AI Agents” under KNOWLEDGE & AI → AI Hub now links to `/admin/ai/agents` so the AI Agents page opens within the admin layout; the main app route `/ai-agents` is unchanged.

- **Admin sidebar fixes:**
